## 3. MECANICAS VIRAIS - O QUE FUNCIONA

### 3.1 Modelo Spotify Wrapped (REFERENCIA #1)

O Spotify Wrapped e o caso mais bem-sucedido de conteudo viral auto-gerado:

**Numeros:**
- **500 milhoes de compartilhamentos** no 1o dia (2025) - +41% vs ano anterior
- **200 milhoes de usuarios engajados** em 24 horas
- **+21% em downloads** do app na 1a semana apos lancamento

**Por que funciona:**
1. **Dados personalizados** - cada pessoa recebe algo unico
2. **Cards visuais bonitos** - cores vibrantes, design bold, pronto pra Story
3. **Formato slideshow** - tap forward no estilo Instagram Stories
4. **Identidade + pertencimento** - "sou parte do fenomeno" + "sou unico nele"
5. **FOMO** - quem nao compartilha fica de fora da conversa
6. **Zero esforco pra compartilhar** - botao direto pro Instagram/WhatsApp

**APLICACAO PARA MISSAO MAIO AMARELO:**
> Criar "cards" personalizados com o resultado do quiz/missao no estilo Wrapped.
> Cada card = imagem pronta pra screenshot/compartilhamento com visual impactante.

---

### 3.2 Modelo BuzzFeed Personality Quiz (REFERENCIA #2)

**Numeros:**
- **96% dos participantes completam** o quiz (taxa absurda)
- Quizzes tem **51.968 shares** no Facebook vs 15.527 de artigos normais (~3.3x mais)
- "What City Should You Actually Live In?" teve **20 milhoes de views**
- GothRider (quiz de motociclista!) gerou **75.000 leads** com **30% de taxa de cadastro**
- Headlines podem aumentar trafego em ate **500%**

**Formulas que funcionam:**
- "Qual ____ voce e?" (50% dos quizzes top usam esse formato)
- Resultados SEMPRE positivos (ninguem compartilha resultado ruim)
- Resultados apelam para ego e vaidade
- 5-15 perguntas (sweet spot)
- Respostas visuais com imagens
- Linguagem casual e acessivel

**Psicologia por tras:**
- Efeito dotacao (endowment effect) - "meu resultado e especial"
- Vies de confirmacao - "sabia que eu era assim!"
- Validacao social - "olha quem eu sou"
- Auto-expressao - o quiz diz algo sobre mim

**APLICACAO PARA MISSAO MAIO AMARELO:**
> Quiz "Qual Piloto de Moto Voce E?" com perfis como "Cauteloso", "Veterano",
> "Protetor", "Lider". Resultado sempre positivo mas com dica de seguranca.

---

### 3.3 Conteudo que gera SCREENSHOT + COMPARTILHAMENTO no WhatsApp

**Gatilhos psicologicos para compartilhamento:**

1. **Emocoes de alta excitacao** - Espanto, medo, diversao, raiva (medo funciona MUITO para seguranca)
2. **Moeda social** - "Compartilhar isso me faz parecer esperto/consciente/responsavel"
3. **Identidade** - "Isso diz algo sobre quem eu sou"
4. **Prova social** - "Todo mundo ta fazendo"
5. **FOMO** - "Nao quero ficar de fora"
6. **Preservacao emocional** - Screenshot = bookmark emocional

**O que faz o dedo ir no botao de compartilhar:**
- Resultado personalizado com nome/foto
- Card visual bonito e legivel (funciona no thumbnail do WhatsApp)
- Score/ranking que da orgulho
- Badge/conquista que parece exclusiva
- Dado chocante que surpreende
- Desafio para o amigo ("sera que voce consegue?")

### 3.4 Gamificacao para Seguranca - Exemplos Reais

**Exemplos Brasileiros de Transito:**

| Plataforma | Mecanica | Resultado |
|------------|----------|-----------|
| **Mob+** (Detran AL) | Quiz + videos + IA + desafios semanais + pontos | Maior percepcao de risco |
| **Jogo das Palavras** (Detran PR) | Palavras-cruzadas; erro remove item de seguranca do ciclista | Consequencia visual |
| **Transito Legal** (Detran CE) | Quiz, caca-palavras, verdadeiro/falso, acerte a placa | Engajamento educativo |
| **Ecovias SP** | Simulador VR de acidente de moto em rodovia | Experiencia imersiva |

**Exemplos de Gamificacao em Seguranca do Trabalho (internacionais):**
- Micro-quiz mobile com leaderboard: **completion rates subiram drasticamente, acidentes caem ~20%**
- Scratch-off gamecards: recompensa por reportar near-miss
- VR safety scenarios com quiz: **~20% reducao em acidentes**
- SC Training: swipe, match tiles, scratch cards para revelar respostas

## 4. ELEMENTOS INTERATIVOS MOBILE - IDEIAS CONCRETAS

### 4.1 Gestos Mobile que Engajam

| Gesto | Uso no App | Exemplo |
|-------|-----------|---------|
| **Tap** | Selecao de resposta, avancar | "Toque para revelar" |
| **Swipe horizontal** | Navegacao entre cards/perguntas | Estilo Tinder (esquerda/direita) |
| **Swipe vertical** | Scroll storytelling | Dados aparecendo conforme desce |
| **Long press** | Revelar informacao oculta | "Segure para ver o que aconteceu" |
| **Shake** | Acao surpresa/easter egg | "Sacuda o celular para embaralhar" |
| **Timer/Countdown** | Urgencia nas respostas | "Voce tem 5 segundos para decidir" |
| **Drag & drop** | Montar equipamento de seguranca | "Arraste o capacete para o piloto" |

**Dado importante:** Conteudo interativo gera **52,6% mais engajamento** que estatico, e usuarios interagem por **2x mais tempo**.

### 4.2 Instagram Stories Interativo

- Polls aumentam engajamento em **+30%**
- Stories interativos: **+50% de engajamento**
- **500 milhoes** de usuarios diarios em Stories
- Contas com alto engajamento em Stories = **1,6x mais visitas ao perfil**

## 5. IDEIAS CONCRETAS PARA "MISSAO MAIO AMARELO"

### IDEIA 1: Quiz "Qual Piloto de Moto Voce E?"
**Mecanica:** 7-10 perguntas sobre habitos no transito
**Formato:** Card visual, tap para avancar (estilo Stories)
**Resultados (todos positivos):**
- "Piloto Guardiao" - prioriza equipamento
- "Piloto Estrategista" - planeja rota com cuidado
- "Piloto Lider" - influencia outros a serem seguros
- "Piloto Veterano" - experiencia e cautela
- "Piloto Alerta" - sempre atento ao redor

**Viralidade:** Card de resultado com visual bold + botao "Compartilhe no WhatsApp" + "Desafie um amigo"

---

### IDEIA 2: "Missao Seguranca" - Desafio Diario
**Mecanica:** 5 missoes durante maio, cada uma com mini-desafio
- Missao 1: "Check do Capacete" - checklist interativo de seguranca
- Missao 2: "Rota Segura" - marcar sua rota mais perigosa
- Missao 3: "Reacao Rapida" - mini-game de tempo de reacao
- Missao 4: "Codigo do Piloto" - juramento interativo
- Missao 5: "Embaixador" - compartilhar e convidar 3 amigos

**Gamificacao:** Badges por missao completada, progresso visual, ranking

---

### IDEIA 3: "Seu Trajeto em Numeros" (Estilo Spotify Wrapped)
**Mecanica:** Usuario responde perguntas sobre sua rotina (km/dia, horas pilotando, etc)
**Output:** Cards personalizados mostrando:
- "Voce percorre X km/mes. Isso equivale a Y viagens SP-RJ"
- "Em X horas no transito, voce fica exposto a Z situacoes de risco"
- "Motociclistas como voce representam X% das vitimas"
- "Seu score de seguranca: 7/10 - Acima de 60% dos pilotos"

**Viralidade:** Cards prontos para screenshot, dados personalizados

---

### IDEIA 4: "Escolha Seu Caminho" - Historia Interativa
**Mecanica:** Choose-your-own-adventure onde cada decisao impacta o desfecho
**Cenarios:**
- Aceitar ou recusar mais uma corrida cansado?
- Atender o celular enquanto pilota?
- Usar ou nao o equipamento completo?
- Cortar pelo corredor ou esperar?

**Desfechos:** Mostram consequencias reais baseadas em dados
**Viralidade:** "Eu sobrevivi todas as 5 decisoes! E voce?"

---

### IDEIA 5: "Tempo de Reacao" - Mini-Game
**Mecanica:** Tap rapido quando obstaculos aparecem (carro abrindo porta, pedestre, buraco)
**Dados reais:** Mostra que a X km/h, 0.5s de atraso = Y metros a mais
**Score:** Tempo medio de reacao + comparacao com "piloto ideal"
**Viralidade:** "Meu tempo de reacao: 0.3s - Mais rapido que 78% dos pilotos!"

---

### IDEIA 6: "Antes de Sair" - Checklist Gamificado
**Mecanica:** Drag-and-drop items de seguranca no piloto (capacete, luva, jaqueta, bota)
**Twist:** Timer + items errados misturados + pontuacao
**Dado real:** "40% dos motociclistas mortos nao usavam capacete"
**Viralidade:** Badge "Equipado Completo" para compartilhar

---

### IDEIA 7: Scrollytelling de Impacto
**Mecanica:** Scroll vertical com dados aparecendo dramaticamente
**Sequencia:**
1. "Quanto tempo voce gasta pilotando por dia?" (input)
2. Scroll -> dados nacionais aparecendo
3. "33 motociclistas morrem POR DIA no Brasil"
4. Contador ao vivo incrementando
5. "Mas 90% dos acidentes sao evitaveis"
6. Call-to-action para o quiz

## 7. STACK TECNICA SUGERIDA

Para o projeto Next.js existente (`ifood_demo`):

| Feature | Tecnologia |
|---------|-----------|
| Quiz interativo | React state machine + Framer Motion |
| Cards compartilhaveis | html-to-image ou @vercel/og (OG image generation) |
| Animacoes de scroll | Intersection Observer + CSS animations |
| Gestos (swipe/tap) | react-swipeable ou use-gesture |
| Timer/countdown | requestAnimationFrame |
| Share WhatsApp | `whatsapp://send?text=` deep link |
| Share generico | Web Share API (`navigator.share()`) |
| Shake detection | DeviceMotionEvent API |
| Imagem do resultado | Canvas API ou SVG para gerar card |
| Analytics | Vercel Analytics ou Umami |
